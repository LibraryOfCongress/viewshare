Supported data file types

Currently you can upload and share:

  • XML MODS
  • Simple rectangular spreadsheets in MS Excel format (Note: the proprietary data format makes them impossible to fully support some Excel functionality)
  • OAI (Beta)
  • ContentDM (Beta)

Supported browsers

Microsoft Internet Explorer v.7 & 8, Firefox v.3.x and Safari v.3.x and v.4.x. Other browsers that mostly work are: Opera v.9.x, Camino v.1.6.x, Firefox v.2.x, but you may encounter issues.




More details are available in User Guide Data File Section or the Troubleshooting Guidelines
