Format your Map data
Before you can design your map, we need to convert your data into latitude, longitude coordinates (e.g. "40.1583, -83.074"). Please select the location related properties in your data (e.g. City, State, Zip) to be converted.

NOTE: If you already have a property that has latitude, longitude values (e.g. "40.1583, -83.074") and would like to bypass this step, use the editor to set the appropriate property type to 'location'.
