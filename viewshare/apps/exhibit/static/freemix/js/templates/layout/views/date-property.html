Format your data
Before you can design your timeline, we need to convert your data into a standardized date format (e.g. ISO 8601 "2014-03-17"). Please select the date / time related property in your data to be converted.



NOTE: If you already have a property that has standardized date values (e.g. ISO 8601 "2014-03-17" year-month-day) and would like to bypass this step, use the editor to set the appropriate property type to 'date/time'.
